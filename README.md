# Portfolio — Muhammad Rakha

Portfolio elegan, simple, dan responsif dengan animasi halus.

## Edit cepat
- Ubah email & tautan sosial di bagian Kontak pada `index.html`.
- Ganti inisial avatar ("MR") di bagian Hero jika punya foto profil sendiri.
- Tambahkan prestasi/portfolio lain dengan menyalin elemen `.card` pada bagian Prestasi.

## Struktur
- `index.html` — markup utama
- `styles.css` — styling (dark, modern, smooth)
- `script.js` — animasi reveal, smooth scroll, auto-year footer

## Jalankan
Cukup buka `index.html` di browser. Tidak perlu build.

## Kustomisasi lanjutan
- Warna utama: atur di `:root` pada `styles.css` (`--brand`, `--brand-2`).
- Tipografi: file menggunakan Google Font Inter.
- Komponen dapat di-duplicate sesuai kebutuhan.
